## Chatbot Bengkel Mobil D.Maju Jaya
Chatbot Bengkel D. Maju Jaya merupakan Chatbot berbasis Telegram yang digunakan untuk memberikan informasi layanan bengkel, harga produk dan jasa otomotif, serta melakukan pencatatan keluhan pelanggan secara otomatis berdasarkan pertanyaan atau keluhan yang diberikan oleh pengguna.

Chatbot ini dikembangkan sebagai bagian dari tugas akhir dan diimplementasikan menggunakan pendekatan pemrosesan bahasa alami (Natural Language Processing) dengan memanfaatkan model IndoBERT yang telah melalui proses fine-tuning untuk menjawab pertanyaan terkait layanan dan harga otomotif secara kontekstual.

### Fitur Utama
- Menyediakan informasi layanan bengkel
- Menyediakan informasi harga oli
- Menyediakan informasi harga barang dan jasa kendaraan
- Mencatat keluhan pelanggan dan membuat kartu pekerjaan

### Cara Menggunakan Chatbot
- Buka aplikasi Telegram lalu cari username "@D_MajuJayaBot" / Melalui website (https://web.telegram.org/k/#@D_MajuJaya_Bot)
- Ketik "/start" untuk memulai Chatbot
- Pilih opsi 1-5 tergantung kebutuhan
- Masukkan pertanyaan / keluhan sesuai kebutuhan 
- Chatbot akan memberikan jawaban

### Teknologi yang Digunakan
- Python
- Hugging Face BERT Base (https://huggingface.co/cahya/bert-base-indonesian-1.5G)
- Model NLP IndoBERT
- SQLite3
